The Silverarch Repository is a 
specialized package repository 
for the Silverarch Linux distribution.

## Installation
1- Enter to your [pacman]( https://wiki.archlinux.org/title/Pacman) config if you use Arch Linux , ArchARM or another Arch-Based Distro

```bash
sudo nano /etc/pacman.conf 
```
2-Put this in bottom of your config 

```bash
[silverarch-repo]
SigLevel = Optional TrustAll
Server = https://raw.githubusercontent.com/iyed-one-st/silverarch-repo/main/x86_64/
```

## Update Pacman

Put this command to update all your repos and your pkgs

```bash
sudo pacman -Syy
```
