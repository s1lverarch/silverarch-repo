# Security Policy

## Reporting Vulnerabilities

We take the security of Silverarch Linux and its associated packages very seriously. If you believe you have found a security vulnerability in any package within this repository, please report it privately.

**Do not open a public issue** for security vulnerabilities, as this could expose the system to exploitation before a fix is available.

### How to report a vulnerability:
Please send an email to: **s1lv3rcore@proton.me**

Your report should include:
- A clear description of the vulnerability.
- Steps to reproduce the issue.
- Any potential impact or exploit scenarios.

We will acknowledge your report within 48 hours and work with you to resolve the issue as quickly as possible.

## Disclosure Policy
We follow a responsible disclosure process. We ask that you keep the vulnerability details confidential until a patch has been released and users have had a reasonable amount of time to update their systems.

Thank you for helping keep Silverarch Linux secure for everyone.
